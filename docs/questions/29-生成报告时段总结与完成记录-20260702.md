# 29 · 生成报告：时段总结与完成记录

**日期**：2026-07-02  
**类型**：新功能 + 产品调性

## 需求

- 生成报告支持两种模板：**时段总结**、**完成记录**
- 项目基调偏二次元、轻松愉快，面向用户**自己记录生活**，文案不要官腔

## 实现

### 模板

| ID | 名称 | 数据来源 |
|----|------|----------|
| `period-summary` | 时段总结 | `period_summaries` 时段 AI 小结 |
| `activity-log` | 完成记录 | 时间线 + 应用时长 + 键鼠/文件小数据 |

### 后端

- `hanpet/src-tauri/src/report/mod.rs`：组装数据；有文本模型时走 AI + 人设，否则本地 Markdown 拼装
- `prompts/report-period-summary.md`、`report-activity-log.md`：轻松手账风格提示词
- `db/reports.rs`：`generated_reports` 表持久化
- IPC：`report_generate`、`report_list`、`report_delete`

### 前端

- `ReportGeneratePage`：两模板卡片、日期范围、「生成小记」、右侧预览
- `HistoryReportsPage`：列表 + 详情 + 删除
- `App.tsx` 等多处副标题/帮助文案改为生活记录口吻

### 使用

1. 侧栏 **生成报告** → 选模板 → 选日期 → **生成小记**
2. 配置文本模型 + 人设后，AI 会润色；未配置则用本地数据拼成
3. **历史报告** 可回看、删除
