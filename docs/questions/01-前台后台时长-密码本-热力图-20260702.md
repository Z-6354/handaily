# 01 · 前台/后台时长分离、应用显示名、三日热力图、密码本

**日期**：2026-07-02  
**状态**：已实现

## 问题

1. 后台记录时间与前台时间分开计算：前台统计当前页面使用，后台统计开始到结束的时间
2. 应用名友好显示（如 `msedge` → Microsoft Edge）
3. 时段热力图改为前天/昨天/今天三条，颜色按使用时长深浅
4. 参考 [小黑日报助手官网](https://www.xiaoheiribao.com/?s=bilibili2#features) 优化功能描述
5. 新增密码本，可选主密码，API 密钥加密存储

## 实现摘要

### 前台 vs 后台时长

- **前台有效时长**（`foreground_ms`）：聚合器统计非 idle 的前台应用切换片段累计时长
- **后台采集时长**（`background_ms`）：`tracking_sessions` 表记录采集开关的墙钟时间（开始到结束）
- 切换采集时通过 `AppState::set_tracking_enabled` 开启/关闭 session

相关文件：`src-tauri/src/db/sessions.rs`、`src-tauri/src/state.rs`、`src-tauri/src/ipc/commands.rs`

### 应用显示名

- `src-tauri/src/tracker/display_name.rs` 将 exe 名映射为友好名称（`msedge` → `Microsoft Edge`）
- 概览、应用排行、时间线均使用 `display_name`

### 三日热力图

- `query_three_day_heatmap()` 返回前天/昨天/今天，每 3 小时一个 slot
- slot 值为该时段**前台使用时长**（ms），颜色深浅按相对时长比例

### 密码本（Vault）

- 表：`vault_config`、`vault_entries`
- 有主密码：AES-256-GCM + PBKDF2（600_000 迭代）
- 无主密码：Windows DPAPI 本机保护
- IPC：`vault_setup/unlock/lock/list/add/update/delete/get_secret`
- UI：侧边栏「密码本」→ `VaultPanel.tsx`

### 前端

- `TodayDashboard`：四指标概览、三日热力图、前台应用排行
- Hero 文案对齐官网：本地存储、直连 AI、密钥加密
