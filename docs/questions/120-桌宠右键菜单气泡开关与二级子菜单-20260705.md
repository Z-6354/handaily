# 桌宠右键菜单：气泡开关与二级子菜单

**日期**：2026-07-05  
**分类**：桌宠 / UI

## 需求

1. 右键菜单新增**气泡台词**开关  
2. **切换模型**、**切换性格**移入二级子菜单（flyout），主菜单更紧凑

## 实现

### 气泡开关

- 设置键 `pet_bubble_enabled`（默认开启，`0` 为关）
- IPC：`pet_get_bubble_enabled` / `pet_set_bubble_enabled`
- 关闭时：前端不显示气泡（含点击/随机台词），并清除当前气泡；后端定时 remark 与 `emit_remark` 均跳过
- `PetStatusPayload` / `PetConfigPayload` 增加 `bubble_enabled` 字段

### 二级菜单

- 主菜单项「切换模型 ›」「切换性格 ›」，点击展开右侧 flyout（窗口右缘不足时向左展开）
- 模型/性格列表仍在 flyout 内动态渲染，选择后关闭整单并执行切换

### 样式

- `pet.css`：`pet-menu-row`、`pet-menu-switch`、`pet-menu-submenu`、`pet-menu-flyout`

## 验证

1. 右键打开菜单 → 气泡开关可切换，关闭后不再弹出台词  
2. 点击「切换模型」→ 二级列表出现，选模型后重建桌宠  
3. 小窗口靠右时 flyout 向左展开不裁切
