# 130 · 人物页搜索与收藏

**日期**：2026-07-07

## 需求

人物列表右上方：搜索（关键词、拼音）+ 收藏；收藏置顶；可筛选「只看收藏」。

## 实现

- `hanpet/src/lib/characterRoster.ts`：`pinyin-pro` 拼音匹配 + 字段关键词；收藏优先排序
- `hanpet/src/hooks/useCharacterFavorites.ts`：设置项 `character_favorites`（JSON id 列表）持久化
- `PersonaPanel.tsx`：右上搜索框 +「收藏」筛选；卡片右上角 ☆/★ 切换
- 样式：`persona-roster-tools`、`persona-card-fav-btn` 等

收藏顺序：新收藏插入最前；列表中收藏人物排在非收藏之前。
