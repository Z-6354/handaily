# 85 · AI 人设 Wiki 导入与仅保留柴郡内置

**日期**：2026-07-04  
**类型**：功能优化

## 需求

1. 用户可通过 Wiki 链接添加人设：自动爬取、清洗资料后走原 AI 流水线生成 Skill。
2. 删除除柴郡外的其他内置性格（小寒助手、菲比、穹）。

## 实现

### 内置人设精简

- `personas/manifest.json` 仅保留 `cheshire`（柴郡），`default_id` 改为 `cheshire`。
- 删除 `default` / `phoebe` / `sora` 的 `.md` / `.json` 源文件。
- `persona/mod.rs`：`EMBEDDED_PERSONAS` / `EMBEDDED_PROFILES` 仅柴郡；启动时 `migrate_legacy_personas` 从 manifest 移除旧预设，若当前选中为已删 ID 则回退到柴郡。

### Wiki 导入人设

- `pet/wiki_scrape.rs` 新增 `extract_persona_reference`：提取角色资料段落 + BWIKI 舰船台词，组装为参考文本（≤24000 字）。
- 新 IPC：`persona_import_wiki` — 爬取 → 清洗 → `persona_process_reference`（预处理 JSON → 生成 Skill → 写入 manifest）。
- 未填人设 ID 时按角色名 `suggest_persona_id` 自动生成（中文名用哈希 ID）。

### 前端

- `PersonaTextImportForm`：Wiki / 粘贴文本 Tab；Wiki 模式默认，支持 BWIKI 链接一键创建/更新。
- `PersonaPanel` 新增卡片文案更新；移除已删预设的 accent 色。

## 使用

1. 设置 → AI 配置 → 配置**思考模型**。
2. AI 人设 → 新增角色 → **Wiki 导入** → 粘贴如 `https://wiki.biligame.com/blhx/柴郡` → **从 Wiki 创建**。
3. 已有自定义人设可在详情页「导入资料」Tab 用 Wiki 更新。
