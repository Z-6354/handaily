# 桌宠动作配置、编辑范围与布局缓存

**日期**: 2026-07-03  
**主题**: 编辑范围（占用窗口 + 滚轮缩放）、点击动作、随机待机动作、开发者功能区、布局持久化

## 需求摘要

1. **编辑范围**：右键菜单进入，拖动右下角调整桌宠窗口占用空间；编辑模式下滚轮缩放角色大小。
2. **点击动作**：用户可配置单击时播放的 Spine 动作；未配置时自动匹配 tap/click 类动作。
3. **随机动作**：待机循环期间，在随机间隔播放用户勾选的动作表中的动作，间隔可配置（min~max 秒）。
4. **UI 重组**：待机动作、点击动作、随机动作表、导入/删除模型移至桌宠页下半部分「开发者功能」。
5. **布局缓存**：位置（已有）、窗口宽高、角色缩放统一持久化到 settings。

## 实现要点

### 后端（Rust）

- `PetAnimationMeta` 扩展：`click_animation`、`random_animations`、`random_min_sec`（默认 30）、`random_max_sec`（默认 120）。
- 新 IPC：`pet_set_click_animation`、`pet_set_random_animations`、`pet_save_layout(width, height, scale)`。
- `pet_save_layout` 同时写入 `pet_width`、`pet_height`、`pet_scale`。
- 同步动作列表时 `prune_animation_refs` 清理无效引用。

### 桌宠窗口（`src/pet/`）

- **编辑范围**：`exitEditBounds` 调用 `pet_save_layout` 保存窗口尺寸与当前 `stageScale`。
- **滚轮缩放**：编辑模式下 `wheel` 事件调整 `stageScale`（0.4~1.5）。
- **SpinePet**：`configureAnimations` 支持点击/随机动作；随机定时器在动作完成后重新调度。

### 设置页（`PetPanel.tsx`）

- 上半：启用、显示、模型、省电、角色大小滑条、气泡频率。
- 下半「开发者功能」：待机、点击、随机动作勾选、频率输入、导入、删除。

## 设置键

| 键 | 说明 |
|---|---|
| `pet_x` / `pet_y` | 窗口位置 |
| `pet_width` / `pet_height` | 占用窗口大小 |
| `pet_scale` | 角色缩放 |
| `pet_anim_meta_{model_id}` | 动作元数据 JSON |
