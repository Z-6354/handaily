# 39 · 预设人设样式优化与详情查看

**日期**：2026-07-02  
**类型**：UI / 人设

## 需求

优化 AI 人设页样式；重写预设人设（默认、柴郡、菲比、空等）；每个人设可**查看详情**，展示 Skill Markdown 与结构化 JSON 资料。

## 实现

### 预设资料

每人设一对文件：

| 文件 | 内容 |
|------|------|
| `personas/{id}.md` | 注入 AI 的 Skill 文档（语气、禁忌、输出格式等） |
| `personas/{id}.json` | 结构化资料（介绍、性格、台词、禁忌、extra） |

内置 JSON 嵌入 `hanpet/src-tauri/src/persona/mod.rs`（`EMBEDDED_PROFILES`），首次启动若磁盘无 `{id}.json` 则自动 seed。

### 后端

- `PersonaInfo.has_profile`：列表是否含结构化资料
- `persona_get_detail` → `PersonaDetail`：`skill_md` + `profile_json` + 元信息
- 人设工坊「应用到 AI 人设」时同步写入 `{id}.json`

### 前端

- **卡片**：顶栏主题色、选用 / 查看详情分离；`has_profile` 显示「含结构化资料」
- **右侧详情面板**（`PersonaDetailPanel`）：三 Tab — Skill 文档 / 结构化资料 / JSON
- **布局**：`.persona-layout` 左列表 + 右 sticky 详情；窄屏单列

### 主题色

| id | 色 |
|----|-----|
| default | `#64748b` |
| cheshire | `#f59e0b` |
| phoebe | `#a78bfa` |
| sora | `#94a3b8` |

## 使用

1. 侧栏 → **AI 人设**
2. 卡片点 **查看详情** → 右侧面板切换 Skill / 结构化 / JSON
3. **选用** 与查看独立，不影响当前选中人设
