# 09 · 今日工作与时间线 UI 对齐小黑日报

**日期**：2026-07-02  
**类型**：前端 UI 优化  
**参考**：小黑日报助手截图（今日工作、工作时间线）

## 问题

在已有绿色主题与分页导航基础上，继续对照小黑日报两张核心页面截图优化：

1. **今日工作**：Hero 标语 + 隐私标签、工作概览三指标、内嵌 24h 时段记录热力条  
2. **工作时间线**：纵向时间轴、卡片式 AI 摘要、工作类型彩色标签、复制日志、时段筛选

## 实现

### 今日工作 `TodayDashboard.tsx`

- Hero：`IconCat` +「写日报，原来可以这么轻松」+ 三条隐私 pill  
- **工作概览**单卡：AI/规则叙述 + 记录条数 / 专注时长 / 主要工作  
- 内嵌 `TodayHeatmap`（今日单行 24 格，少→多图例，悬停显示时段摘要）

### 工作时间线 `TimelineView.tsx`

- 弃用表格，改为 `.vtimeline` 纵向绿线 + 时间点 + 卡片  
- 合并 `period_summaries`、`activity_insights` 与 segment 生成卡片文案与标签色  
- 工具栏：复制日志到剪切板 + 近30分/1h/2h/今天筛选  
- 编辑/删除按钮占位（disabled，后续迭代）

### 新增组件

- `src/components/TodayHeatmap.tsx` — 今日页专用紧凑热力条

### 样式 `styles.css`

- `.hero-panel--today`、`.work-overview-*`、`.today-heatmap-*`  
- `.vtimeline-*`、`.wt-tag`、`.pulse-icon`

### `App.tsx`

- `TodayDashboard` 传入 `heatmap`、`workTypes`  
- 移除今日页未使用的 `metrics` 拉取
