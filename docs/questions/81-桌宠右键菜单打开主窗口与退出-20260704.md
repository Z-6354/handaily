# 81 — 桌宠右键菜单：打开主窗口与退出

**日期**: 2026-07-04  
**主题**: 右键「打开小寒日报」无效；退出未实现

## 现象

- 桌宠右键菜单点「打开小寒日报」主窗口不出现
- 菜单无「退出」，仅「隐藏桌宠」

## 根因

1. **打开主窗口**：桌宠窗口 `always_on_top`，`pet_open_main` 仅 `show+focus` 主窗口，主窗实际在桌宠后面，用户以为没打开。
2. **退出**：托盘有 `app.exit(0)`，桌宠菜单从未接入。

## 修复

**Rust `pet/mod.rs`**

- `show_main_window`：打开主窗前取消桌宠置顶、`unminimize`、show、focus；可选 `main-navigate` 事件
- `restore_pet_topmost_if_visible`：主窗失焦/隐藏后恢复桌宠置顶

**`ipc/commands.rs`**

- `app_exit`：`app.exit(0)`
- `pet_open_main(page?)` 走 `show_main_window`

**`lib.rs` / `tray.rs`**

- 主窗 `CloseRequested` / `Focused(false)` 时恢复桌宠置顶
- 托盘「显示主窗口」与左键切换改用 `show_main_window`

**前端**

- `pet/main.ts`：菜单新增「退出」；菜单 `mousedown` 防冒泡；操作失败 toast；聚焦桌宠时 `setAlwaysOnTop(true)`
- `App.tsx`：监听 `main-navigate` 切换侧栏页

**权限**：`app-commands.toml` 增加 `app_exit`

## 验证

重新编译运行后：

1. 右键桌宠 →「打开小寒日报」→ 主窗口应出现在桌宠前方
2. 右键 →「退出」→ 进程结束、托盘消失
3. 双击桌宠同样可打开主窗口
