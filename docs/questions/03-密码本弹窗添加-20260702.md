# 03 · 密码本改为弹窗添加，字段简化为名称/密钥/网址

**日期**：2026-07-02  
**状态**：已实现

## 需求

密码本改为弹窗添加；属性包含名称、API 密钥；可选所在网址。

## 实现

- 新增 `VaultEntryModal.tsx`：居中弹窗，字段为名称（必填）、API 密钥（添加必填 / 编辑可留空不修改）、所在网址（可选）
- `VaultPanel`：列表页「添加」按钮打开弹窗；编辑同样走弹窗；移除页内 inline 表单
- 列表列：名称、所在网址（可点击外链）、密钥、操作
- 后端 `VaultEntryInput` / `VaultEntryPublic`：`website_url` 替代原 `provider` + `note`（DB `note` 列存网址）
- 编辑时 `secret` 为空则仅更新名称与网址，不重加密密钥
