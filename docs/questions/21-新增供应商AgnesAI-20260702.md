# 21 · 新增供应商 Agnes AI（agens）

**日期**：2026-07-02  
**类型**：功能增强

## 需求

新增 OpenAI 兼容供应商 **Agnes AI**（内部 id：`agens`）。

## 配置

| 字段 | 值 |
|------|-----|
| id | `agens` |
| 名称 | Agnes AI |
| Base URL | `https://apihub.agnes-ai.com/v1` |
| API 风格 | `openai` |

## 实现

- `VendorId::AGENS` + `default_vendors()` 条目
- `migrate_vendors` 改为补全所有缺失的默认供应商（老用户重启后自动出现 Agnes AI）
- `urls.rs` 单元测试覆盖 Agnes 路径拼接

## 使用

1. 在 [Agnes 平台](https://platform.agnes-ai.com/) 创建 API Key
2. 密码本保存密钥 → 设置 → AI 配置 → Agnes AI 卡片关联
3. 「测试」或「导入模型」；若列表为空可手动添加（如 `agnes-2.0-flash`）
