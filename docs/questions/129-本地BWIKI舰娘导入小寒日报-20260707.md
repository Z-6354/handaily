# 129 · 本地 BWIKI 舰娘导入小寒日报

**日期**：2026-07-07

## 需求

从 `mcp/blhx-wiki` 本地 SQLite 导入舰娘到 HANDAILY，流程与 Wiki 导入相同（本地解析 reference → AI 生成 Skill），仅数据源从联网改为本地库。

## 实现

- `persona/import_reference.rs`：共用 AI 导入流水线
- IPC：`persona_import_blhx_local`（参数 `wikiTitle`）
- 前端：`PersonaTextImportForm` 新增「本地 BWIKI」Tab
- CLI：`cargo run --bin blhx_import -- "欧根亲王,贝尔法斯特,..."`

环境变量 `BLHX_WIKI_DB_PATH` 指向 `mcp/blhx-wiki/data/blhx.sqlite`。

## 试导入 5 角色

已执行：

```
欧根亲王, 贝尔法斯特, 胡德, 能代, 拉菲
```

写入 `%AppData%/xiaohan-daily/data/personas/`，manifest 已更新；人物列表通过 `resolve_all_characters` 自动出现（默认皮肤 model 为 chaijun 除非有同名模型）。

## 使用

**CLI（开发者批量）**

```powershell
$env:BLHX_WIKI_DB_PATH="D:\0HAN\HANDAILY\mcp\blhx-wiki\data\blhx.sqlite"
cd src-tauri
cargo run --bin blhx_import -- "舰娘名1,舰娘名2"
```

**应用内**：人物 → 新增 → 本地 BWIKI → 输入舰娘名 → 从本地库创建。
