# 31 · 工作时间线 AI 简介

**日期**：2026-07-02  
**类型**：新功能

## 需求

时间线每条记录的「简介」改为 AI 生成，语气随人设，类似时段小记风格。发送给模型的信息包括应用、窗口标题、混合分析洞察等。

## 实现

### 数据注入（user prompt）

每条片段 JSON 含：

- 时间段、应用名、窗口标题、时长
- `insights`：截图/文本混合分析结果
- `period_hint`：重叠的时段 AI 小结（如有）

**system prompt**：当前激活人设 Markdown（`PreparedTextChat` 自动注入）

### 后端

- `prompts/timeline-segment-describe.md`
- `timeline/mod.rs`：plan → batch AI（每批 ≤20）→ 缓存
- `db/timeline_cache.rs`：按 `started_at|ended_at|agg|title` 缓存，避免重复调用
- IPC：`timeline_describe(date?, limit, offset)` — 释放 DB 锁后再 `run_async`

### 前端

- `TimelineView`：加载时间线后调用 `timelineDescribe`，卡片展示 AI 简介与工作类型标签
- 无模型时回退本地简介（洞察 / 时段 hint / 应用+窗口模板）
