# 开机自启动 ACL 权限缺失修复

**日期**：2026-07-05  
**分类**：设置 / 启动 / Tauri

## 现象

设置 → 登录时自动启动 切换失败：

```
Command autostart_set_enabled not allowed by ACL
```

## 根因

Tauri 2 能力集 `allow-app-commands` 白名单未包含 `autostart_get_status` / `autostart_set_enabled`，invoke 被 ACL 拦截。

## 修复

在 `src-tauri/permissions/app-commands.toml` 的 `commands.allow` 中添加上述两条 command。

- `src-tauri/permissions/app-commands.toml`
