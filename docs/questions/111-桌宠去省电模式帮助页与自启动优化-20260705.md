# 桌宠去省电模式、帮助页反馈入口与自启动优化

**日期**：2026-07-05  
**分类**：桌宠 / 帮助 / 启动

## 需求

1. 去掉桌宠「省电模式」设置（极简静态图模式存在 bug）
2. 帮助页挂上 B 站主页，说明 bug 可在视频评论或私信反馈
3. 优化开机自启动选项

## 处理

### 省电模式

- `PetPanel.tsx` 移除省电模式下拉与概览 badge 中的模式文案
- `pet/mod.rs` 的 `model_power_mode` 固定返回 `balanced`，忽略历史 DB/meta 中的 minimal/full

### 帮助页

- `HelpGuideGrid.tsx` 新增「反馈与联系」区块，链接 [B 站主页](https://space.bilibili.com/146915875)（显示名：**万年烟火**）
- 文案：视频下方评论或私信反馈

### 自启动

- 注册表写入 `"exe" --tray`，登录启动时不弹主窗口（release 默认隐藏；debug 下 `--tray` 也不 show）
- 设置页文案改为「登录时自动启动」，说明后台静默、托盘常驻、桌宠可选显示
- 切换成功后 toast 提示
