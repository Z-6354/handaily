# 25 · 火山引擎默认模型配置

**日期**：2026-07-02  
**类型**：配置增强

## 需求

火山引擎 Agent Plan 默认模型：

- 文本：`ark-code-latest`
- 多模态：`doubao-seedream-5.0-lite`

## 实现

### `config/vendors.json`

供应商 `volcano` 增加 `default_models` 字段。

### `catalog.rs`

- `VendorDefaultModels` 结构
- 加载时从内置 JSON 补丁合并 `default_models`（旧用户本地副本也能获得）

### `config.rs`

- `models_for` 将目录默认模型置于列表首位
- `apply_vendor_defaults`：选中火山且模型为空时自动填入默认 ID
