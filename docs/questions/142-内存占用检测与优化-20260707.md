# 142 · 内存占用检测与优化

**日期**：2026-07-07

## 优化项

### Rust

- **Roster 缓存**：`Arc<Vec<CharacterMeta>>` + manifest mtime 指纹，翻页不再重复读盘合并
- **分页筛选**：`roster_filter_indices` 只索引，不 clone 866 条 meta
- **列表轻量 brief**：不读 persona JSON（trait 用 description 摘要）；不查 wiki 图鉴 URL
- **图鉴按需 SQL**：移除整表 `CATALOG_MAP` 预加载，改为单条查询 + 复用连接
- **内存统计 IPC**：`app_memory_stats` → roster 缓存人数 + 本地头像文件数

### 前端

- **移除全量回退**：分页失败仅重试，不再 `charactersList` 加载 866+ 条
- **头像懒加载**：IntersectionObserver，可见时才触发 `characters_cache_avatar`
- **桌宠 blob LRU**：最多 8 个 blob URL，淘汰时 `revokeObjectURL`

### UI

人物页顶栏显示：`缓存 N 人 / M 头像`
