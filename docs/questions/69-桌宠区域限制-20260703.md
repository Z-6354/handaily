# 桌宠区域限制（禁止跳出桌面）

## 需求

桌宠窗口不得拖出/缩放到虚拟桌面可见区域外。

## 实现

### Rust（`pet/mod.rs`）

- `virtual_screen_bounds` + `clamp_pet_position`：按虚拟桌面 clamp，留 8px 边距
- `save_position` / `show_pet` / `save_layout`：读写时 clamp
- `attach_position_guard`：监听 `WindowEvent::Moved`，拖拽过程中实时拉回
- `set_pet_position`：程序设位时带 suppress，避免与 guard 递归
- IPC：`pet_get_screen_bounds`、`pet_save_position` 返回 clamp 后坐标

### 前端（`pet/main.ts`）

- 缓存 `pet_get_screen_bounds`
- `clampWindowPosition` / `clampWindowBounds`：编辑范围缩放时限制
- `savePosition`：应用 Rust 返回的 clamp 坐标
