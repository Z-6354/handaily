# 107 · 主窗口隐藏停轮询与后台深度休眠

**日期**：2026-07-05  
**标签**：性能、轮询、懒加载、后台线程

## 改动

### 主窗口（`App.tsx`）
- 刷新前 `getCurrentWindow().isVisible()`：主窗口隐藏（托盘常驻）时 **跳过全部 IPC 轮询**。
- 窗口重新 `tauri://focus` 时立即刷新一次。
- **懒加载** 时间线、热力图、应用记录页，进一步缩小首包。

### 后台线程
- **poller**：采集暂停时 sleep 2s → **5s**，减少空转唤醒。
- **period_scheduler ticker**：采集暂停时 tick 30s → **120s**，且跳过 `check_triggers`。

## 预期

- 只开桌宠/托盘、主窗口隐藏时，前端不再每 5s 打 3～6 次 IPC。
- 暂停采集后 CPU 唤醒频率再降一档。
- 首屏 JS 再减（时间线等大页按需加载）。
