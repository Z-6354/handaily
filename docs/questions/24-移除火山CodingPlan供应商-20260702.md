# 24 · 移除火山引擎 Coding Plan 供应商

**日期**：2026-07-02  
**类型**：配置精简

## 需求

删除供应商「火山引擎 Coding Plan」（`volcano_coding`）。

## 实现

- `config/vendors.json`：移除 `volcano_coding` 条目，加入 `removed_vendor_ids`
- `catalog.rs`：加载时合并内置 `removed_vendor_ids` 并过滤供应商列表（旧用户本地 `vendors.json` 仍含该条目也会自动隐藏）
- `StoredAiState::migrate`：选中 Coding Plan 的用户会切回默认供应商并清理相关模型

## 保留

- **火山引擎 Agent Plan**（`volcano`）
