# 40 · 人设页标签化布局与导入文件

**日期**：2026-07-02  
**类型**：UI / 人设

## 需求

- 「选择人设」单独占一页（标签）
- 去掉「人设文件」路径展示
- 去掉「人设工坊」
- 新增「导入文件」标签
- 点击「查看详情」切换到「人设详情」标签，而非右侧侧栏

## 实现

### 页面结构（左侧导航）

| 标签 | 内容 |
|------|------|
| 选择人设 | 卡片网格、选用、测试 |
| 人设详情 | Skill / 结构化 / JSON；可下拉切换角色 |
| 导入文件 | 选择 `.md` / `.json` 导入 |

### 导入

- 前端 `PersonaImportPanel`：多文件选择、队列预览
- 后端 `persona_import`：同名 stem 合并为同一人设 ID，写入 `{id}.md` / `{id}.json` 并更新 `manifest.json`

### 移除

- `CharacterBuilderPanel` 不再挂载于 AI 人设页
- `app_get_personas_path` 展示区块移除（IPC 仍保留）

## 使用

1. **选择人设** → 选用 / 查看详情  
2. **查看详情** → 自动切到 **人设详情** 标签  
3. **导入文件** → 选 `角色.md` + 可选 `角色.json` → 导入到人设列表
