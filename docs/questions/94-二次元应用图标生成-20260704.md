# 二次元应用图标生成

**日期**：2026-07-04  
**分类**：UI / 品牌

## 需求

根据「小寒日报」名称与轻二次元/手账风格，生成应用图标并替换 Tauri 打包图标。

## 风格参考

- 品牌色：薰衣草紫 `#7c8aff`、粉 `#ff9eb5`、天蓝 `#7dd3fc`
- 主题：日报助手 + 桌宠伴侣，手账/数据记录感
- 元素：Q 版猫耳少女、笔记本柱状图、雪花（小寒）

## 产出

| 文件 | 用途 |
|------|------|
| `assets/xiaohan-daily-icon-source.png` | 1024 源图 |
| `hanpet/src-tauri/icons/32x32.png` | 托盘/小尺寸 |
| `hanpet/src-tauri/icons/128x128.png` | 标准图标 |
| `hanpet/src-tauri/icons/128x128@2x.png` | 256 HiDPI |
| `hanpet/src-tauri/icons/icon.ico` | Windows exe/安装包 |
| `public/app-icon.png` | 主界面侧栏品牌图 |

## 应用

重新 `npm run tauri:build` 后 exe 与安装包将使用新图标；侧栏品牌区同步显示新图标。
