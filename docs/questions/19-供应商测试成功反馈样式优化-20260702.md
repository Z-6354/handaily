# 19 · 供应商测试成功反馈样式优化

**日期**：2026-07-02  
**类型**：UX 优化

## 问题

测试成功时标题与详情重复出现「连接成功」，且保存模型信息堆在同一段落里，可读性差。

## 实现

### `parseTestSuccess`

- 去掉消息前缀「连接成功，」及尾部保存统计句
- 括号内说明拆到 `hint`（如 `/models 不可用，建议使用 ark-code-latest`）
- 使用 `imported_text` / `imported_vision` 生成 `tags`：`2 文本`、`0 多模态`

### `SettingsFeedbackBanner`

- 成功态标题行右侧展示统计标签
- 左侧绿色竖条 + 渐变背景

### 展示效果

```
✓ 连接成功    [2 文本] [0 多模态]
  Agent Plan 密钥有效
  /models 不可用，建议使用 ark-code-latest
```
