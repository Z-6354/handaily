# 20 · 移除供应商内置模型，仅导入与自定义

**日期**：2026-07-02  
**类型**：行为变更

## 需求

- 供应商不再提供默认/内置模型列表
- 模型全部来自「导入模型」或「手动添加」
- 用户自定义模型需持久化到本地

## 实现

### 后端

- 删除 `builtin_models` 及 Plan 套餐的 `builtin_remote` 回退
- `models_for` 仅返回 `custom_models` + `imported_models`
- 默认 `text_model` / `vision_model` 为空
- `sanitize_model_selection`：加载时清除不在已保存列表中的旧内置选中项
- `ai_add_custom_model`：写入 `custom_models` 并自动设为当前选中模型
- `ai_import_models`：空列表时报错，提示手动添加
- AI 调用前校验模型 ID 非空

### 前端

- 无模型时下拉框显示「请先导入或手动添加」并禁用
- 导入/切换供应商后自动选中首个可用模型
- 选项标注「(自定义)」或「(已导入)」
