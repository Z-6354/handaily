# 16 · 测试/导入模型自动保存本地

**日期**：2026-07-02  
**类型**：功能增强

## 需求

- 导入模型后自动保存到本地
- 测试供应商连通时顺带导入并保存模型

## 实现

### 存储

`AiConfig.imported_models`（SQLite `ai_config` JSON），`models_for` 合并：自定义 → 已导入 → 内置。

### 后端

- `persist_imported_models()` — 统一落盘
- `ai_import_models` — 拉取后写入 `imported_models`
- `ai_test_vendor` — 测试成功后拉取文本/多模态模型并保存，消息含「已保存 N 个文本 / M 个多模态模型到本地」

### 前端

- 导入/测试成功后 `aiGetConfig` + `loadModels` 刷新下拉框
