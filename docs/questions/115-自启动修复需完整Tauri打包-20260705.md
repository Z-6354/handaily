# 自启动修复需完整 Tauri 打包说明

**日期**：2026-07-05  
**分类**：构建 / 启动

## 现象

- 直接运行 `target\release-fast\xiaohan-daily.exe` 打不开或行为异常
- 开始菜单/桌面快捷方式（旧安装包）未更新，自启动 ACL 修复未生效

## 根因

此前仅用 `cargo build --profile release-fast` 编译 Rust，**未执行 `tauri build`**：

- 未嵌入最新前端 `dist`
- 未生成 NSIS 安装包，已安装版快捷方式仍指向旧 exe
- Tauri 权限 manifest 等需完整打包链路

## 正确更新方式

```powershell
cd D:\0HAN\HANDAILY
$env:FORCE_FE_BUILD="1"
npm run tauri:build
```

产物：

| 用途 | 路径 |
|------|------|
| 便携 exe | `hanpet/src-tauri\target\release-fast\xiaohan-daily.exe` |
| 安装包（更新快捷方式） | `hanpet/src-tauri\target\release-fast\bundle\nsis\小寒日报_0.1.0_x64-setup.exe` |

**推荐**：运行 NSIS 安装包覆盖安装，再从开始菜单启动；或先托盘退出旧进程后双击新 exe。
