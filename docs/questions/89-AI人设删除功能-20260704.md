# 89 · AI 人设删除功能

**日期**：2026-07-04  
**类型**：功能新增

## 需求

用户需要删除自定义人设，但 UI 无删除入口。

## 实现

- 后端 `persona::delete_persona` + IPC `persona_delete`
- 删除 manifest 条目及 `personas/{id}.md`、`.json`
- 若删除当前使用人设，自动切回默认（柴郡）
- **内置人设（柴郡）不可删除**
- 前端：选角卡片底部「删除」、详情页顶栏「删除」

## 使用

AI 人设 → 自定义角色卡片或详情页 → **删除** → 确认。
