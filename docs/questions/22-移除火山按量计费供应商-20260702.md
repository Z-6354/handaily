# 22 · 移除火山方舟按量计费供应商

**日期**：2026-07-02  
**类型**：配置精简

## 需求

删除供应商「火山方舟（按量计费）」(`volcano_ark`)。

## 实现

### 移除

- `VendorId::VOLCANO_ARK` 与 `default_vendors()` 中的按量计费条目
- `providers.rs` / 前端错误提示中与按量计费相关的分支

### 迁移（`migrate_vendors`）

启动加载配置时自动：

1. 从 `vendors` 列表移除 `volcano_ark`
2. 若文本/多模态供应商选中按量计费 → 切换为 **Agent Plan**（`volcano`），并清空对应模型选中
3. 删除该供应商下的 `custom_models` / `imported_models`
4. 变更写入 SQLite

### 保留的火山供应商

- 火山引擎 Agent Plan（`/api/plan/v3`）
- 火山引擎 Coding Plan（`/api/coding/v3`）
