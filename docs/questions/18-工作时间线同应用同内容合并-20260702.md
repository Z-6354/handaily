# 18 · 工作时间线同应用同内容合并

**日期**：2026-07-02  
**类型**：功能优化

## 需求

工作时间线中，多个同一软件、同一窗口内容的记录应合并为一条，避免重复展示。

## 实现

### 后端 `hanpet/src-tauri/src/db/stats.rs`

- 新增 `merge_timeline_segments()`：按时间升序遍历，相邻且 `app_name + window_title` 相同的段合并
- 合并后累加 `duration_ms`，保留最早 `started_at`、最晚 `ended_at`
- `query_timeline` 先拉取当日全部符合条件的段，合并后再分页；`total` 为合并后条数

### 前端

- `TimelineView` 时间轴显示起止时间范围（如 `10:00:00 – 10:12:00`）及总时长
- 空状态与页面副标题说明「相邻同应用同窗口会自动合并」

### 合并规则

- **会合并**：时间相邻、同一应用、同一窗口标题（中间可有被过滤的 idle/短段）
- **不合并**：中间夹了其他应用，或窗口标题不同
