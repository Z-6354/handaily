# 127 · 人物详情 Tab 二次合并

**日期**：2026-07-07  
**标签**：UI、人物、Tab 合并

## 需求

人物详情页进一步合并 Tab：

1. **皮肤 · 模型 + 桌宠 + 导入模型** → 一个 Tab
2. **性格 + 性格资料 + 导入资料** → 一个 Tab

## 改动

详情页主 Tab 现为 **2 个**：

| Tab | 区块（同页滚动） |
|-----|------------------|
| **皮肤 · 桌宠** | 皮肤网格（含「+ 导入皮肤」卡片）→ 桌宠显示 → 动作与台词 |
| **性格** | Skill → 结构化资料 → 导入资料 → JSON → 编辑表单 |

## 127 续 · Tab 三次合并（2026-07-07）

将原独立的 JSON、编辑 Tab 并入「性格」Tab 底部，详情页仅保留 **皮肤 · 桌宠** 与 **性格** 两个主 Tab。

## 127 续 · 导入皮肤并入选择器（2026-07-07）

「导入皮肤模型」从桌宠分区移至**皮肤网格末尾**，样式同人物列表「+ 新增」卡片；点击弹出 `SkinImportModal` 完成 Spine 导入。

`CharacterPetSettings` 改为纵向分区，不再单独占「导入模型」子 Tab。
