# 139 · 人物头像与标签优化

**日期**：2026-07-07

## 需求

- 清理无用前端代码，减轻列表/详情内存占用
- 优化角色卡片标签（阵营/舰种/稀有度，隐藏冗长 `source`）
- 从 BWIKI 图鉴导入头像到小寒日报并展示

## 实现

### 头像存储

`%AppData%/xiaohan-daily/data/characters/avatars/{persona_id}.{jpg|png|webp}`

通过 Tauri `assetProtocol`（`$APPDATA/xiaohan-daily/**`）在前端 `convertFileSrc` 加载；图片 `loading="lazy"`，失败回退名字首字。

### 数据来源

`mcp/blhx-wiki/data/blhx.sqlite` 表 `catalog`：`avatar_url`、`faction`、`ship_type`、`rarity`，按 `display_name` 与人设 `name` 匹配。

### Rust

- 新模块 `src-tauri/src/character/avatar.rs`
- IPC：`characters_import_avatars_batch`（默认每批 50，跳过已有，同步标签）
- CLI：`cargo run --bin avatar_import -- --limit 50` / `--all` / `--force`

### 类型扩展

`CharacterBrief` / `CharacterDetail` / `CharacterMeta` 增加 `faction`、`ship_type`、`rarity`、`avatar_path`。

### 前端

- `CharacterAvatar` 组件 + `characterDisplay.ts`（accent / 标签 / 首字）
- 人物页「导入 BWIKI 头像」按钮
- 删除未使用的 `filterCharacterRoster` / `sortCharactersByFavorite`

## 使用

```powershell
# CLI 批量（推荐先小批测试）
cd src-tauri
cargo run --bin avatar_import -- --limit 50

# 应用内：人物页 → 「导入 BWIKI 头像」，重复点击直到剩余为 0
```
