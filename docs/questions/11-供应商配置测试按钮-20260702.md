# 11 · 供应商配置：密钥左侧测试按钮

**日期**：2026-07-02  
**类型**：功能增强

## 需求

设置页「供应商 API 密钥」区域，在密码/密钥选择框左侧增加「测试」按钮，验证连通性与密钥有效性。

## 实现

### 后端

- `ai/providers.rs`：`test_vendor_connection`
  - Ollama：`GET /api/tags`
  - OpenAI 兼容：`GET /v1/models`（Bearer 密钥）
- IPC：`ai_test_vendor(vendor_id)` → `{ ok, message }`

### 前端

- `SettingsPanel`：每行供应商 `[测试] [密钥下拉]`，下方显示结果（✓/✗）
- Ollama 同样提供测试（测本地服务）
- 未解锁密码本时测试按钮禁用
