# 126 · 人物页与桌宠页合并

**日期**：2026-07-07  
**标签**：UI、人物、桌宠、合并

## 需求

桌宠页和人物页合并；查看人物时可一并查看皮肤、模型、性格、桌宠设置。

## 改动

| 项 | 说明 |
|----|------|
| 侧栏 | 移除独立「桌宠」入口，保留「人物」 |
| `PersonaPanel` | 人物详情 Tab：**皮肤·模型** / **性格** / **性格资料** / **桌宠** / JSON·导入·编辑 |
| `CharacterPetSettings` | 自原 `PetPanel` 抽出：显示、动作、台词、导入模型 |
| `PetPanel` |  deprecated，重导出 `PersonaPanel` |
| 导航兼容 | `main-navigate` 的 `pet` 重定向到 `persona` |

## 人物详情 Tab

| Tab | 内容 |
|-----|------|
| 皮肤 · 模型 | 皮肤选择器（含 model 名称） |
| 性格 | Skill 文档 |
| 性格资料 | 结构化 profile |
| 桌宠 | 启用/大小/气泡、动作、台词、导入 Spine 模型（按当前皮肤 model） |

选用人物后，切换皮肤与桌宠配置才会生效。
