# 隐藏启动时的终端黑窗口

**日期**：2026-07-04  
**分类**：Windows / 体验

## 现象

安装或运行小寒日报时，会弹出一个黑色终端窗口，里面显示注册表 / vision AI 等日志。

## 原因

1. Release 可执行文件未声明 `windows_subsystem = "windows"`，Windows 按控制台程序处理，分配 CMD 窗口。
2. 开机自启动同步调用 `reg.exe` 时未加 `CREATE_NO_WINDOW`，可能短暂闪出控制台。

## 修复

- `hanpet/src-tauri/src/main.rs`：`#![cfg_attr(not(debug_assertions), windows_subsystem = "windows")]`（仅 release 隐藏；debug 仍保留控制台便于开发）
- `hanpet/src-tauri/src/system/autostart.rs`：`reg_command()` 统一设置 `creation_flags(CREATE_NO_WINDOW)`

## 说明

- 直接双击 **release** 的 `xiaohan-daily.exe` 或 NSIS 安装包安装后运行，不应再出现黑窗口。
- `npm run tauri:dev` / debug 版仍可能有控制台，属正常。
- 不要用 `start.bat` 启动已安装版；应从开始菜单或桌面快捷方式启动。

## 验证

重新 `npx tauri build --bundles nsis` 后安装测试。
