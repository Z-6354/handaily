# 141 · 人物列表分页搜索与本地头像缓存

**日期**：2026-07-07

## 需求

- 列表/皮肤加载改为**页码分页**（非无限滚动）
- 搜索按**搜索按钮**触发，支持**搜索历史**
- 卡片标签每排两个：**皮肤** + **AI 性格摘要**（不显示阵营/舰种）
- 头像首次从 Wiki 下载，之后读**应用 data 目录**本地缓存
- 缓存统一在 `%AppData%/xiaohan-daily/data/` 下

## 实现

### 分页

- 人物列表：`Pagination` 组件，每页 24 条
- 皮肤列表：每页 12 条，页码切换

### 搜索

- 输入框 +「搜索」按钮，Enter 提交
- 历史存于 settings `persona_search_history`（应用 data 目录 sqlite）

### 标签

- `trait_summary` 来自 persona profile 性格/说话风格
- CSS `.persona-card-tags` 两列网格

### 头像

- 路径：`data/characters/avatars/{id}.{ext}`
- IPC `characters_cache_avatar` 按需下载
- 前端 `CharacterAvatar` 优先 `convertFileSrc(avatar_path)`
