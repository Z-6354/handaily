# 桌宠设置「立即更新」与状态自动刷新

**日期**: 2026-07-03  
**主题**: 运行状态按钮改为立即更新，点击后无需重启

## 需求

- 「立即显示」改为「立即更新」
- 点击后重新加载桌宠配置并刷新设置页运行状态，无需重启应用

## 实现

`PetPanel.tsx`：

- 新增 `applyPetUpdate()`：`petSetEnabled` / `petShow` → `petReload`（全量重读 config + ViewerEX）→ `refreshStatus()`
- 按钮文案「立即更新」，成功提示「桌宠已更新」
- 启用开关、保存省电/缩放、保存随机动作后均调用 `refreshStatus()` 同步 `petVisible` 等状态

## 验证

1. 修改随机动作或模型后点「立即更新」— 桌宠重载且状态栏同步
2. 无需重启 `start-dev.bat`
